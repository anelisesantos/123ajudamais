Bem-vindo à API de Unidades de Saúde

Esta API fornece acesso a informações sobre unidades de saúde, incluindo detalhes sobre abrangência, endereço, contato e muito mais. Esta API esta disponivel somente para regiao do Distrito Federal

Como Usar a API:

  1. Consultar Unidades de Saúde por Abrangência e UF:

    • Endpoint: https://123ajuda.tech/api/unidades_de_saude?abrangencia={abrangencia}&uf={uf}
    • Método: GET
    • Parâmetros:
      • abrangencia: Tipo de abrangência desejado (por exemplo, "municipal", "estadual")
      • uf: Sigla do estado (por exemplo, "SP", "RJ")

  2. Consultar Detalhes de uma Unidade de Saúde por ID:

    • Endpoint: https://123ajuda.tech/api/unidades_de_saude/id/{id}
    • Método: GET
    • Parâmetros:
      • id: ID único da unidade de saúde desejada

  3. Documentacao Swagger

    • Endpoint: https://123ajuda.tech/apidocs
